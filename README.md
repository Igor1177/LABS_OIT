## Лабораторная работа №2. Арифметические операции с числами.  Системы счисления
### Цель работы.  
**_Изучить внутреннее представления чисел в компьютере._**
### Общее задание.
_Написать программу, позволяющую выполнять  операции для целых чисел и чисел с плавающей запятой.  Задание состоит из двух частей, части А) и части Б). Обе части можно выполнить как в одной программе, так и в двух программах. В программах запрещено использовать специализированные средства языка программирования для отображения двоичных, шестнадцатеричных и восьмеричных целых чисел. Все операции программировать «как вручную». Для проверки работоспособности программ необходимо предоставить не менее 3-х контрольных примеров._


**Важно!  Язык программирования – Си. В программе разрешено использовать библиотеки stdio.h, conio.h, math.h., stdlib.h, string.h. Любые изменения и дополнения принимаются только по согласованию с преподавателем.**

**Вариант задания: 18.   
_А) Умножение 2-х чисел в системе счисления, заданной вариантом. Четверичная. 
Б) Запрограммировать перевод числа с плавающей запятой  в его внутреннее представление в компьютере:  -763,200345. Объем памяти для записи числа - 4 Байта._**

### Список источников

        2. Гашков С. Б. Системы счисления и их применение.  – Москва : МЦНМО, 2004. — 52 с.
        3. Яглом И.М. Системы счисления. — Квант, 1991 № 12, с. 14–22.
