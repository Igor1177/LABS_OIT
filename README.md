## Лабораторная работа №1. Шифрование-дешифорвание
### Цель работы 
Изучить различные методы шифрования, а также получить практические навыки шифрования и дешифрования текстовых данных.
Общее задание 
Написать программу, осуществляющую шифрование и дешифрование каких-либо данных. Для проверки работоспособности программ необходимо предоставить не менее 5-и контрольных примеров. В квадратных скобках указана ссылка на справочный источник.
### Содержание отчёта
1) Титульный лист.
2) Задание к лабораторной работе. Задание по варианту.
3) Теоретическое описание метода шифрования (Описание, формулы, соотношения).
4) Пример. Пошаговый алгоритм шифрования на конкретных данных.
5) Контрольные тесты для программы.
6) Ответы на теоретические вопросы.
7) Список литературы. 
8) Приложение (с нового листа). Код программы.

**Важно!  Язык программирования – Си (С++). В программе разрешено использовать библиотеки stdio.h, conio.h, math.h., stdlib.h, string.h. Любые изменения и дополнения принимаются только по согласованию с преподавателем.**

**Вариант задания: 23. _Тип шифрования – двойной квадрат_**

1. Для каких целей производят шифрование данных?
2. Приведите примеры, не менее трёх, методов шифрования данных, которых нет в вариантах к лабораторным работам.
3. Зашифруйте своё имя и фамилию шифром Цезаря со смещением 7, 11, 31, 57.
